FEATURES
[+] Classic Tetris gameplay
[+] Lightweight binary (14–16 KB)
[+] Bitwise grid storage
[+] Zero external dependencies
[+] Works on any Unix-like terminal

REQUIREMENTS
- Linux, macOS, or Unix-like system
  (Windows users: use WSL)
- GCC or Clang (C99 compatible)

BUILD AND RUN 
Standart build ~ 16kB
git clone https://github.com/yourusername/terminal-tetris.git
cd terminal-tetris
gcc main.c -o tetris
./tetris

Optimized build ~ 14kB
gcc -Os -s -ffunction-sections -fdata-sections -Wl,--gc-sections \
    -fno-unwind-tables -fno-asynchronous-unwind-tables \
    -fno-stack-protector -fomit-frame-pointer -flto \
    -Wl,--strip-all -Wl,--build-id=none \
    main.c -o tetris
HOW TO PLAY 
a  -> move left
d  -> move right
w  -> rotate
Ctrl+C -> quit

HOW IT WORKS 
Each row is stored as a single 16-bit integer. Each bit represents one cell: 0 -> empty, 1 -> block. 
Visual row is ....###...
Stored in memory as: 0000000000111000
